## Source Of Truth

- parameter schema: `src/model/schema.ts`
- mesh generation: `src/febio/mesh/`
- nucleus-cytoplasm interface: `src/febio/interfaces/nucleusCytoplasm.ts`
- FEBio export: `src/febio/export/index.ts`
- FEBio export detachment contract: `src/febio/export/index.ts`
  export bundles declare explicit detachment event and metric ownership for external FEBio results.
- result normalization: `src/febio/import/normalizeFebioResult.ts`
  native-first `localNc` / `localCd` normalization, native face-data fallback, native regional `contactFraction` / `nativeGap` preservation, native `sourceNormal` / `sourceDamage` / `sourceShear` retention, explicit external detachment event normalization, detachment metric supplementation, and imported event backfill across final state and history.
- external FEBio result conversion: `scripts/convert_febio_output.mjs`
  converted FEBio log payloads now emit explicit detachment events and detachment metrics before canonical import, reuse native face tangential traction for shear when those face snapshots expose extra traction columns, and prefer the standard `plotfileSurfaceData.*` contact-traction bridge when that standard bridge payload is present.
- FEBio face-data export metadata: `src/febio/export/index.ts`
  standard export now self-describes that logfile `face_data` currently emits `contact gap;contact pressure`, and also declares a standard plotfile `contact traction` bridge contract for `localNc` / `localCd` tangential observation.
- classification: `src/results/classification.ts`
  native-first detachment judgment with explicit proxy fallback labeling and shared classification application.
- public detachment assessment: `src/public-api.ts`
  exposes canonical `assessDetachment` and `applyRunClassification` so compatibility/runtime code can reuse the same thresholds and classification application path.
- nucleus-cytoplasm interface stabilization: `src/febio/interfaces/nucleusCytoplasm.ts`
  solver-primary sticky cohesive approximation, explicit soft-start stabilization ramp, stabilization validation, and proxy/native observation ownership.

## Compatibility Note

- `src/` is the canonical edit target.
- `generated/dist/` is generated by `node scripts/build-dist.mjs`.
- `generated/febio_exports/` is the default location for FEBio export outputs.
- `tmp/logs/` is the default location for disposable local logs.
- `index.html` now loads `generated/dist/browser/main.js`.
- The current browser runtime still boots the legacy compatibility bundle (`simulation.js`, `js/simulation-febio.js`, `js/simulation-ui.js`) through that compiled entry while the source-of-truth modules are moved into `src/`.
- Classification callers in the legacy runtime now prefer the canonical public API classifier bridge, including shared classification application, before falling back to compatibility-local logic.
- Lightweight compatibility runtime now emits explicit `detachmentStart` / `detachmentComplete` events using canonical public API detachment assessment when available.

- Keep `README.md`, `CODEBASE_STRUCTURE.md`, and `PROGRESS.md` aligned.
