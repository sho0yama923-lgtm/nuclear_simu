# Incidents and Root Causes

This document stores durable lessons that are retired from `PROGRESS.md`.

## Purpose

- Keep `PROGRESS.md` focused on the active milestone.
- Move resolved major causes and prevention rules here.
- Keep transient run logs out of this file unless they explain a reusable failure mode.

## Entry format

Each entry should include:

- Status
- Discovered in
- Symptom
- Cause
- Impact
- Fix or mitigation
- Regression guard
- References
